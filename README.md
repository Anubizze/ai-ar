# Корпоративный сайт мукомольного завода

Современный фронтенд корпоративного сайта мукомольного предприятия. Разработан по референсу [makfa.ru](https://www.makfa.ru/).

## Возможности

- **Публичный сайт**: главная, о заводе, партнёры, сертификаты, контакты
- **Каталог продукции**: мука, отруби, крупы с фильтрацией
- **Модуль заявок**: форма оптового заказа в два шага
- **Адаптивная вёрстка**: корректная работа на ПК, планшетах и смартфонах
- **SEO**: семантическая разметка, мета-теги

## Технологии

- React 18 + Vite
- React Router 6
- Tailwind CSS 3

## Запуск

```bash
cd frontend
npm install
npm run dev
```

Сайт откроется по адресу http://localhost:5173

## Сборка

```bash
npm run build
```

Результат в папке `dist/`.

## Структура проекта

## Маршруты

| Путь | Описание |
|------|----------|
| `/` | Главная |
| `/about` | О заводе |
| `/partners` | Партнёры |
| `/certificates` | Сертификаты |
| `/contacts` | Контакты |
| `/catalog` | Каталог продукции |
| `/catalog/:category` | Каталог по категории |
| `/product/:id` | Карточка товара |
| `/order` | Оформление заявки |
